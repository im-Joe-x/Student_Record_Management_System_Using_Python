# Student_Record_Management_System_Using_Python
A Python-based Student Record Management System that allows users to add, display, search, and delete student records while automatically calculating grades through a menu-driven console application.

Project Purpose

This project was developed to understand the fundamentals of Python programming by creating a practical Student Record Management System. It demonstrates how programming concepts such as functions, lists, dictionaries, loops, and conditional statements can be used to build a real-world application for managing student information.
